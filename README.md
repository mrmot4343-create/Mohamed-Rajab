# Financial Statements Streamlit App

تطبيق **Streamlit** بسيط لإدارة دليل الحسابات، ميزان المراجعة، وإظهار القوائم المالية الأساسية مع تحليل مالي مختصر.

## المتطلبات
- Python 3.9+
- مكتبات: `streamlit`, `pandas`, `xlsxwriter`

يمكن تثبيت الحزم المطلوبة بالأمر:

```bash
pip install streamlit pandas xlsxwriter
```

## طريقة التشغيل
1. شغّل التطبيق:

```bash
streamlit run financial_app.py
```

2. استخدم الشريط الجانبي للتنقل بين الصفحات:
   - إعدادات الشركة (اسم، نوع، فترة، شعار)
   - دليل الحسابات (إضافة/تعديل/حذف الحسابات)
   - ميزان المراجعة (إدخال أرصدة المدين والدائن)
   - القوائم المالية (قائمة الدخل والميزانية العمومية مع إمكانية تصدير Excel)
   - تحليل مالي (هوامش الربح ونسبة الديون إلى الأصول)

> يمكن تعديل دليل الحسابات الافتراضي حسب نوع الشركة (خدمية أو تجارية) واستخدامه كأساس للتقارير.
